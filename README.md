# KHUMU FRONTEND

`Next.js`를 사용하여 쿠뮤 어플리케이션의 웹뷰를 `Server Side Rendering`(이하 SSR)을 통해 띄운다.

## SSR vs Client Side Rendering(이하 CSR)

|         구분          |     SSR     |     CSR     |
| :-------------------: | :---------: | :---------: |
| SEO(검색 엔진 최적화) |      O      |      X      |
| 초기 컨텐츠 로딩속도  | 비교적 빠름 | 비교적 느림 |
