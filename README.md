## SSR vs Client Side Rendering

|         구분          |     SSR     |     CSR     |
| :-------------------: | :---------: | :---------: |
| SEO(검색 엔진 최적화) |      O      |      X      |
| 초기 컨텐츠 로딩속도  | 비교적 빠름 | 비교적 느림 |
